# Habito — Habit Tracker App

تطبيق تتبع عادات كامل، متصل بباك اند حقيقي (Supabase)، جاهز للنشر كـ PWA وبعدها Google Play.

## 📁 هيكل المشروع

```
habito/
├── index.html              الصفحة الرئيسية
├── manifest.json           إعدادات PWA
├── sw.js                   Service Worker (تخزين أوفلاين)
├── css/
│   └── styles.css          كل التنسيقات (ثيمات، مودات، مكونات)
├── js/
│   ├── config.js           رابط ومفتاح Supabase
│   ├── constants.js        الفئات، الثيمات، اللغات، مكتبة الإيموجي
│   ├── icons.js             مجموعة أيقونات SVG
│   ├── i18n.js              الترجمات (5 لغات) + دوال مساعدة
│   ├── data.js               طبقة الاتصال بـ Supabase + حسابات الـ streaks/XP
│   ├── ui-core.js            التوست، المودالز، التنقل، تأثيرات بصرية
│   ├── render.js             عرض كل الصفحات (الرئيسية/التقدم/العادات/الإعدادات)
│   ├── modals.js             نوافذ إضافة عادة/الحساب/الشروط/النسخ الاحتياطي

│   ├── terms-content.js       نص الشروط والخصوصية

└── assets/
    ├── logo.png               اللوجو اللي بعتّه
    └── icons/                 أيقونات PWA بأحجام مختلفة
```

## 🐛 الأخطاء اللي اتصلحت

### 1) "إضافة عادة مش بتشتغل" + "مش بيرضى يدخل تاني بنفس الإيميل"
**السبب الحقيقي**: النسخة اللي فاتت كانت بتستخدم `window.storage` (API خاص بمعاينة Claude بس) عشان تحفظ جلسة الدخول. لما رفعتي التطبيق على GitHub Pages، الـ API ده مش موجود خالص هناك — فجلسة الدخول كانت بتفشل تتحفظ من غير أي رسالة خطأ، فأي طلب بعد أول واحد كان بيتبعت من غير تسجيل دخول حقيقي، والـ Row Level Security في قاعدة البيانات كان بيرفضه بصمت.

**الحل**: شلت الاعتماد على `window.storage` خالص، ودلوقتي بيستخدم `localStorage` العادي بتاع المتصفح (الافتراضي في Supabase) — وده صح 100% لأي موقع حقيقي منشور.

### 2) الأخطاء الصامتة
أي عملية فاشلة (حفظ، إضافة، حذف) دلوقتي بتظهر رسالة توست واضحة للمستخدم بدل ما تفشل من غير ما حد يعرف.

## ✨ حاجات جديدة

- **صفحة الحساب**: تعديل الاسم، رفع صورة بروفايل حقيقية (متخزنة في Supabase Storage)، عرض الإيميل
- **الشروط والخصوصية**: صفحة كاملة (عربي/إنجليزي، والباقي بالإنجليزي كاحتياطي)
- **نسخ احتياطي واستعادة**: تصدير كل بياناتك كملف JSON، واستعادتها وقت ما تحبي
- **هوية Habito**: اللوجو، الألوان، 4 ثيمات (Forest / Ocean / Sunset / Midnight) مبنية على نفس نظام الألوان اللي بعتّيه
- **تصميم أغنى**: خلفية متدرجة قوية + كرة زجاجية مضيئة حوالين دائرة التقدم + رسم بياني بتدرج لوني للـ Progress Flow

## ⚠️ ملاحظات مهمة

- **حذف الحساب**: الزرار موجود في الواجهة، لكن الحذف الفعلي لحساب Supabase محتاج صلاحية خاصة (service role) متسمحش تتحط في كود الفرونت إند لأسباب أمنية. لو عايزة تفعّليه فعليًا، محتاجة Edge Function بسيطة على Supabase — قوليلي لو عايزاني أعملها.
- **الشروط والخصوصية**: النص ده نموذج بداية معقول مش استشارة قانونية. قبل ما تنشري فعليًا على المتجر، يفضل تراجعيه مع محامي.
- **الترجمة**: عربي وإنجليزي كاملين 100%. فرنساوي وألماني وصيني: واجهة التطبيق مترجمة كاملة، لكن نص الشروط بالإنجليزي كاحتياطي.

## 🔄 آخر تحديث (الجولة الثانية)

**أخطاء اتصلحت:**
- **رفع صورة البروفايل**: كانت بتفشل بصمت مع بعض صيغ الصور (زي HEIC من الآيفون). دلوقتي أي صورة بتتحول تلقائيًا لـ JPEG مربّع 480×480 قبل الرفع — بيشتغل مع أي صيغة وحجمه أصغر وأسرع.
- **نقط الاحتفال (confetti) اللي مش بتختفي**: ضفت تنظيف احتياطي (safety-net) يشيلها حتى لو الأنيميشن الأساسي فشل.
- **رسم Progress Flow مش ظاهر**: السبب كان إن الـ canvas معندوش ارتفاع ثابت في الـ CSS، فـ Chart.js كان بينهار لحجم صفر. اتصلح بحاوية بارتفاع ثابت.

**تعديلات الواجهة:**
- صفحة الإعدادات بقت أبسط: الثيمات واللغة دلوقتي في زرارين منفصلين بيفتحوا نافذة لكل واحد، بدل ما يكونوا ظاهرين طول الوقت
- زرار "Edit Profile" اتشال، ودلوقتي أيقونة القلم اللي على صورة البروفايل هي بس اللي بتفتح تعديل الحساب
- ضفت حقل "الجنس" (ذكر/أنثى) في تعديل البروفايل
- ضفت شريط "أكثر العادات استخدامًا" تحت دائرة التقدم في الرئيسية
- مكتبة الإيموجي اتوسّعت لـ 9 تصنيفات وأكتر من 180 أيقونة

**نظام Premium:**
- ثيمين جديدين فاخرين (Aurora و Gold) مقفولين خلف صفحة دفع (paywall)
- صفحة الدفع فيها قائمة ميزات (ثيمات حصرية، عادات من غير حد أقصى، إحصائيات متقدمة، تذكيرات مخصصة، بروفايل من غير علامة هابيتو)
- ⚠️ **مهم**: زرار "Upgrade" دلوقتي بيعمل فتح تجريبي (demo unlock) بس — مفيهوش بوابة دفع حقيقية لسه، لأن ده محتاج ربط فعلي بـ Stripe (للويب) أو Google Play Billing (بعد النشر على المتجر). لو عايزة تفعّلي دفع حقيقي، قوليلي أساعدك تختاري وتظبطي أي واحدة فيهم.
- ألوان الثيمات المجانية الأربعة اتظبطت كمان لتناسق أفضل (كل ثيم دلوقتي بينتقل بين لونين بتدرج حقيقي بدل لونين متشابهين).
